# CLAUDE.md

## 项目概述
创建一个待办网站，可以设置代办事项等功能

## 技术栈
- 前端React + TypeScript +Vite
- Tailwind CSS （样式）
- Zustand （状态管理）
- date-fns（日期处理）
- LocalStorage（数据存储）

## 开发规范
- 使用函数式组件 + Hooks，禁用 class 组件
- 所有样式使用 Tailwind CSS，不单独写 CSS 文件
- 组件保持可复用，props 类型用 TypeScript 明确定义
- 关键逻辑需添加中文注释

## 数据模型
- id(唯一标识)
- title(标题)
- description(描述)
- category(分类，有工作、学习、生活、娱乐四个选项)
- priority(优先级:低、中、高)
- dueDate(截止日期)
- completed(是否完成)
- createdAt(创建时间)

## 设计规范
- 浅色主题：背景 `#0a0a0a`，主文字 `#ffffff`
- 强调色使用渐变（如紫色 → 蓝色 / 青色 → 紫色）
- 所有页面需移动端适配（mobile-first）
- 图片使用懒加载（loading="lazy"）

## 页面功能清单
- 页面左侧上部分是统计（统计总任务，待完成，已完成，完成率（已完成除以总任务））
- 左侧下部分是是设置标题、描述、截止日期、优先级、分类。
- 右侧上面分类筛选，样式是一行四个筛选项：1.筛选优先级 2.状态（已完成，待完成）3.分类（工作，学习，生活，娱乐）4.搜索（根据标题关键词搜索事项）
- 右侧下面是已经创建的列表。每行列表显示标题、简介、时间、状态、优先级，右侧有编辑和删除按钮。
- 刷新页面不会丢失。

## 注意事项
- 保持设计简洁，不过度设计
- 确保所有链接可点击，外链用 `target="_blank" rel="noopener noreferrer"`
- 数据统一维护在 `src/data/` 目录，不硬编码在组件里
